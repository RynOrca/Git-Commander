# Git Commander 验收检查清单

## 项目初始化检查

- [ ] 项目结构符合 Electron + React + TypeScript 架构
- [ ] package.json 包含所有必要依赖
- [ ] TypeScript 配置正确，无编译错误
- [ ] Vite 配置正确，能正常启动开发服务器
- [ ] Electron 主进程能正常启动
- [ ] 主进程与渲染进程 IPC 通信正常
- [ ] 日志系统正常工作

## Git服务层检查

- [ ] simple-git 库正确集成
- [ ] 能成功打开本地Git仓库
- [ ] 能成功创建新Git仓库
- [ ] 能获取文件状态（未跟踪、已修改、已暂存）
- [ ] 能暂存单个文件
- [ ] 能暂存所有文件
- [ ] 能取消暂存文件
- [ ] 能执行提交操作
- [ ] 能克隆远程仓库
- [ ] 能推送到远程仓库
- [ ] 能从远程仓库拉取
- [ ] 能获取远程更新
- [ ] 能获取分支列表
- [ ] 能创建新分支
- [ ] 能切换分支
- [ ] 能合并分支
- [ ] 能撤销文件修改
- [ ] 能获取提交日志

## UI组件检查

- [ ] 应用窗口正确显示（标题栏、关闭/最小化/最大化按钮）
- [ ] 工具栏按钮正确显示
- [ ] 文件列表正确显示各类状态文件
- [ ] 文件状态用不同颜色区分（未跟踪=灰、已修改=黄、已暂存=绿）
- [ ] 提交信息输入框可用
- [ ] 分支列表正确显示
- [ ] 提交历史正确显示

## 用户交互检查

- [ ] 打开仓库对话框正常工作
- [ ] 创建仓库对话框正常工作
- [ ] 所有Git操作执行前显示确认对话框
- [ ] 操作成功显示Toast通知
- [ ] 操作失败显示友好错误提示
- [ ] Loading状态正确显示
- [ ] Ctrl+O 快捷键能打开仓库
- [ ] Ctrl+Enter 快捷键能提交
- [ ] Ctrl+R 快捷键能刷新

## 稳定性检查

- [ ] 应用启动时间小于3秒
- [ ] 大仓库（1000+提交）操作不卡顿
- [ ] Git操作超时正确处理
- [ ] 异常错误不导致应用崩溃
- [ ] 窗口关闭时正确退出应用

## 打包检查

- [ ] 能执行生产构建
- [ ] 生成的 .exe 文件能正常运行
- [ ] 应用图标正确显示
- [ ] 应用名称和版本正确
